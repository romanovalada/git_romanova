Техническое задание на внедрение информационной системы "Hangul" для обучения корейскому языку

1.	ОБЩИЕ СВЕДЕНИЯ
1.1.	Полное наименование ИС: Обучающая система обучения корейскому языку "Hangul "  
1.2.	Шифр темы: HKL-25
1.3.	Предприятие-разработчик системы: LanguageTech Solutions  
1.4.	Предприятие-заказчик системы: Центр современных языков "ЛингваМир"  
1.5.	Система создается на основании технического задания (ТЗ): Настоящего документа  
1.6.	Плановый срок начала работ: 2025-09-01  
1.7.	Плановый срок окончания работ: 2025-12-17  
1.8.	 Автоматизируемая система создается на модульной основе с использованием облачных технологий. 
1.9.	Оформления и предъявления заказчику результатов работы 
определяется после получения начальной версии продукта, в которой должны быть реализованы все основные функции, определенные в ТЗ. Предусматривается поэтапная сдача модулей системы с проведением приемочных испытаний для каждого модуля.

2.	НАЗНАЧЕНИЕ И ЦЕЛИ СОЗДАНИЯ СИСТЕМЫ

2.1.	Вид автоматизируемой деятельности: Обучение корейскому языку, изучение алфавита Хангыль, развитие навыков чтения, письма и произношения.  

2.2.	 Перечень автоматизируемых процессов:
- Интерактивное изучение корейских букв (согласных, гласных, двойных согласных) 
- Тренировка правильного произношения с использованием аудиозаписей носителей языка 
- Контроль прогресса обучения 

2.3.	Наименование и значение показателей, которые будут достигнуты в результате внедрения ИС:
- Увеличение скорости изучения алфавита Хангыль на 40% по сравнению с традиционными методами
- Сокращение времени на подготовку преподавателей на 25%
- Возможность изучения языка в любом месте и в любое время (офлайн-доступность) 

3.	ХАРАКТЕРИСТИКИ ОБЪЕКТА АВТОМАТИЗАЦИИ

3.1.	Краткие сведения о предприятии: 
Центр современных языков "ЛингваМир" основан в 2010 году и специализируется на преподавании восточных языков. В центре обучаются более 1000 студентов ежегодно, работают 15 преподавателей. В настоящее время обучение корейскому языку проводится с использованием разрозненных материалов и приложений (Duolingo, Memrise, Teuida ), что не позволяет обеспечить единую систему контроля прогресса и адаптивную программу обучения. 

3.2.	Существующие проблемы:
- Отсутствие единой системы управления процессом обучения
- Отсутствие персонализации обучения под нужды каждого студента
- Сложность отслеживания прогресса учащихся для преподавателей

4.	ТРЕБОВАНИЯ К ИС

4.1.	Требования к системе в целом

4.1.1.	Требования к функциям:
- Система должна предоставлять интерактивные уроки по изучению букв Хангыль (согласных, гласных, двойных согласных)
- Система должны предоставлять обучению новых слов в каждой сфере
- Система должна работать в офлайн-режиме 

4.1.2.	Требования к надежности:
- Время безотказной работы системы должно составлять не менее 99,5%
- Восстановление после сбоев должно происходить не более чем за 15 минут
- Сохранность данных пользователя должна обеспечиваться при любых условиях

4.1.3.	Требования к безопасности:
- Шифрование персональных данных пользователей
- Аутентификация и авторизация пользователей

4.1.4.	Требования к эргономике:
- Интуитивно понятный интерфейс с поддержкой русского языка
- Минимальное время обучения работе с системой (не более 15 минут)

4.2.	Требования к программному обеспечению ИС

4.2.1.	Технические требования:
- Поддержка офлайн-работы с синхронизацией при появлении соединения

4.2.2.	Требования к интерфейсам:
- Возможность экспорта данных о прогрессе учащихся

4.2.3.	Лингвистические требования:
- Профессиональное озвучивание носителями корейского языка 
- Минимизация использования романизации 

4.3.	Требования к техническому обеспечению АС

4.3.1.	Минимальные требования к клиентским устройствам:
- Операционная система: Windows 10/11 (64-bit)
- Процессор: Intel Core i3 8-го поколения или аналогичный AMD
-Оперативная память: 8 ГБ RAM
-Свободное место на диске: 500 МБ для установки + 2 ГБ для хранения JSON-базы и офлайн-контента
-Графика: DirectX 11 совместимая видеокарта с 2 ГБ памяти
-Звуковая карта: Совместимая с DirectX
-Дополнительно: Микрофон для упражнений по произношению

4.3.2.	Рекомендуемые требования к клиентским устройствам:
-Операционная система: Windows 11 (64-bit)
-Процессор: Intel Core i5 10-го поколения или аналогичный AMD Ryzen 5
-Оперативная память: 16 ГБ RAM
-Свободное место на диске: 1 ГБ для установки + 5 ГБ для JSON-базы и офлайн-контента
-Графика: DirectX 12 совместимая видеокарта с 4 ГБ памяти
-Дополнительно: Качественный микрофон
Обязательно: Микрофон (встроенный или внешний)
